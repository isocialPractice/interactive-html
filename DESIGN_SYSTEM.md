# Interactive HTML Design System

## Color Palette
Inspired by build-electronic-circuits.com and allaboutcircuits.com

### Primary Colors
- **Primary Blue**: `#0066CC` - Main interactive elements, links
- **Primary Green**: `#00A86B` - Success states, positive indicators
- **Dark Blue**: `#004080` - Headers, important text
- **Light Blue**: `#E6F2FF` - Backgrounds, subtle highlights

### Accent Colors
- **Orange**: `#FF6B35` - Warnings, important callouts
- **Yellow**: `#FFB800` - Highlights, active states
- **Red**: `#E63946` - Errors, critical indicators
- **Purple**: `#7209B7` - Special features

### Neutral Colors
- **Background**: `#F5F7FA` - Page background
- **Surface**: `#FFFFFF` - Cards, panels
- **Border**: `#E1E4E8` - Dividers, borders
- **Text Primary**: `#1A1A1A` - Main text
- **Text Secondary**: `#6C757D` - Secondary text
- **Text Muted**: `#ADB5BD` - Disabled/muted text

## Typography
- **Font Family**: 'Inter', 'Segoe UI', -apple-system, system-ui, sans-serif
- **Heading Font Weight**: 600-700
- **Body Font Weight**: 400
- **Monospace**: 'SF Mono', 'Monaco', 'Consolas', monospace

## Spacing Scale
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- 2xl: 48px

## Border Radius
- sm: 4px
- md: 8px
- lg: 12px
- xl: 16px
- full: 9999px

## Shadows
- sm: 0 2px 4px rgba(0,0,0,0.08)
- md: 0 4px 12px rgba(0,0,0,0.1)
- lg: 0 8px 24px rgba(0,0,0,0.12)
- xl: 0 16px 48px rgba(0,0,0,0.15)

## Component Guidelines

### Buttons
- Primary: Blue background, white text, hover state darkens
- Secondary: Gray background, dark text
- Border radius: 8px
- Padding: 12px 24px
- Font weight: 600

### Input Fields
- Border: 2px solid #E1E4E8
- Focus: Border color changes to primary blue
- Border radius: 8px
- Padding: 12px 16px

### Cards
- Background: White
- Border radius: 12px
- Shadow: md
- Padding: 24px

### Icons
- Use simple, line-based icons
- Size: 20-24px for inline, 32-48px for featured
- Color: Match text or use primary color
